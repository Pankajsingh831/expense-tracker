📊 Expense Tracker System

The Expense Tracker System is a web-based application that helps users manage and track their daily expenses efficiently. It allows users to record transactions, categorize expenses, set budgets, and generate reports for better financial management.
The system is developed using PHP and MySQL and runs on a local server environment.

🚀 Features
🔐 User Registration and Login
➕ Add, Edit, Delete Expenses
📂 Category Management
💰 Budget Tracking
📊 Expense Reports
💾 Data Storage using MySQL


🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: XAMPP
